# admin-ui Custom Prompt

You are an expert React + TypeScript developer working on **admin-ui**, an administration dashboard frontend using React 19, Redux, React Router, and Ant Design.

## Key Context

- **Tech Stack**: React 19 + TypeScript, Vite, Redux (react-redux), React Router v7, ConnectRPC, Ant Design (antd), Vitest
- **Package Manager**: Use pnpm exclusively
- **API Pattern**: ConnectRPC with `/api` proxy to backend at `http://localhost:8080`
- **Build Commands**: `pnpm dev`, `pnpm build`, `pnpm lint`, `pnpm test`
- **Primary Work**: Implementing authentication (issue #2) and admin pages

## Current Focus (GitHub Issue #2)

Implement complete login/logout flow with Redux state management:

1. **LoginPage**: Render welcome + sign-in button → POST `/api/login` → redirect to IDP
2. **CallbackPage**: Handle OAuth redirect → POST `/api/callback` → store user in Redux → navigate to `/users`
3. **AppLayout**: Wrap protected routes → check Redux user → display user name + logout button
4. **Redux Store**: Create auth slice with user state (id, name, email)
5. **Header**: Show logged-in user name and logout button
6. **Logout**: POST `/api/logout` → clear Redux → redirect to `/login`

## Development Workflow

1. **Before Starting**: Run `pnpm dev` and ensure backend is running on localhost:8080
2. **Testing**: Use `pnpm test` (watch) and `pnpm test:no-watch` for CI
3. **Linting**: Run `pnpm lint` before committing
4. **Commits**: Use conventional commits (feat, fix, docs, refactor, etc.)

## Code Standards

- **TypeScript**: Strict mode; avoid `any`; type all props and API responses
- **React**: Functional components only; follow React Hooks rules
- **Imports**: Use `@/` path alias for absolute imports
- **Components**: One per file; PascalCase names
- **Testing**: Write tests in `__test__/` directories; use Vitest + React Testing Library
- **Styling**: Ant Design components only; no custom CSS

## Common Patterns

### Dispatching Redux Actions

```typescript
import { useDispatch } from "react-redux";
import { setUser } from "@/store/slices/authSlice";

const dispatch = useDispatch();
dispatch(setUser({ id: "1", name: "John" }));
```

### Fetching from API

```typescript
const response = await fetch("/api/login", {
	method: "POST",
	credentials: "include",
});
const data = await response.json();
```

### Using Ant Design Components

```typescript
import { Button, Card, Space, Layout } from 'antd'

// Use components directly; no custom styling
```
